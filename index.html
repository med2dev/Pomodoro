Instructions:

The Pomodoro method is a time management method developed by Francesco Cirillo. This method centers around time-boxing, with periods of uninterrupted work separated by small, intentional breaks. The traditional Pomodoro is 30 minutes long: 25 minutes of pure work followed by a 5 minute break. After 4 Pomodoros, take a longer 15-30 minute break.

To use this timer, choose a Pomodoro length (the default is 25 min), a break length (default: 5 min), and the number of Pomodoros you would like to complete before taking a longer break (default: 4). When you are ready to begin, click the button labeled Start Pomodoro Session. The timer will continue until the number of Pomodoros you chose completes. If you would like to end the timer yourself, you may do so by clicking the End Pomodoro Session button.

If you would prefer not to limit the number of Pomodoros, remove the default value for Number of Pomodoros. The timer will continue until you click the End Pomodoro Session button.

Cirillo, F. (2006). The pomodoro technique (the pomodoro). Agile Processes in Software Engineering and, 54(2), 35.
